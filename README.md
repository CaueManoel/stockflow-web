# StockFlow Web

Sistema web para **gerenciamento de estoque e venda de produtos**, desenvolvido para facilitar o controle de produtos, organização de inventário e acompanhamento de vendas.

## 📌 Sobre o Projeto

O **StockFlow Web** é uma aplicação que permite gerenciar produtos em estoque e registrar vendas de forma simples e organizada.
O objetivo do projeto é demonstrar conceitos de desenvolvimento web, organização de código e criação de sistemas administrativos.


## ⚙️ Funcionalidades

* 📦 Cadastro de produtos
* 📊 Controle de estoque
* 💰 Registro de vendas
* 🔎 Visualização e gerenciamento de produtos
* 📋 Interface administrativa simples e prática

## 🛠️ Tecnologias Utilizadas

* PHP
* HTML
* CSS
* JavaScript
* MySQL

## 📂 Estrutura do Projeto

```
stockflow-web
│
├── assets/        # Arquivos de estilo, imagens e scripts
├── controller/    # Lógica da aplicação
├── index.php      # Página principal do sistema
└── .htaccess      # Configurações do servidor
```

## 🚀 Como Executar o Projeto

1. Clone o repositório:

```
git clone https://github.com/CaueManoel/stockflow-web.git
```

2. Coloque o projeto dentro da pasta do servidor local (ex: **WAMP, XAMPP ou Laragon**).

3. Configure o banco de dados no MySQL.

4. Acesse no navegador:

```
http://localhost/stockflow-web
```
